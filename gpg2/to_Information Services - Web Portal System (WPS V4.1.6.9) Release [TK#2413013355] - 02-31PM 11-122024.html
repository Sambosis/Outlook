A new portal release WPS V4.1.6.9 has been deployed.  The following fixes and features are included in this release:

 

Highlights

·        The “HOME>CUSTOMER>CUSTOMER” module now include the Google map icon outside of the “ADDRESS INFORMATION” panel.
·        The “HOME>AGREEMENT>ELECTRONIC AGREEMENT MACHINE” module now sends a email to the customer, route supervisor and branch manager when an agreement is rejected.
·        The “HOME>AGREEMENT>ELECTRONIC AGREEMENT MACHINE” module now sets the associated SCM to acquired when the agreement is finalized.
·        The “HRA>ONLINE FEATURE” hamburger menu button now reloads the menu screen when reselecting the “HRA>ONLINE FEATURE” menu button.
 
 

15 additional bug fixes and improvements have been added.  Special thanks to Chris Bengtson, Martin McAuliff, David Giron, Morgan Wilson, Michelle Fortwengler and everyone else involved.
